# RoAutomation Group Finder

## Overview

**RoAutomation Group Finder** is a tool that scans Roblox's API endpoints to search for groups that are **unlocked** and **ownerless**.

> **Note:** This project was never production-ready and remains unfinished. Development was abandoned shortly after it began, and much of the remaining code was AI-generated.

---

## How It Works

The tool operates in two stages:

1. **Group Discovery**
   - Scans Roblox API endpoints to identify existing group IDs.

2. **Ownerless Monitoring**
   - Continuously checks discovered group IDs, looking for groups that become ownerless and are unlocked.
